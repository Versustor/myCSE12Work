Lab 2: Simple Data Path 

-----------
DESCRIPTION

In this lab, the user will select the data source (source select) and the addresses 
of the read and write registers. The user can then set the program to store these 
values to a specific register and if prompted to, perform left arithmetic shift using 
two of these stored values.

-----------
FILES

-
Lab2.lgi

This file includes the simulated circuitry of the sequential logic circuit for the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the Multimedia Logic program (MML). 
Click the run simulation button to activate the simulated circuitry and input switches.
