Lab 1: Intro to Logic Simulation

-----------
DESCRIPTION

In this lab, the user activates simulated switches to send input into their respective logic circuits. 
The first page will visually depict the output of the logic circuits via simulated LEDs. 

-----------
FILES

-
Lab1.lgi

This file includes the simulated circuitry for Part A, B, and C of the lab. 

-----------
INSTRUCTIONS

This program is intended to be run using the Multimedia Logic program (MML). 
Click the run simulation button to activate the simulated circuitry and input switches.
